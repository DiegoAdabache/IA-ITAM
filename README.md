# IA-ITAM
Repositorio para almacenar los proyectos hechos para la materia de Inteligencia Artificial en el ITAM con el profesor Andrés Gómez de Silva.
Se realizó una IA que juega Ultimate Tic Tac Toe (o Gato de Gatos) con una estrategia clara. Para nuestra estructura nos basamos en el 
algorítmo Minimax con poda alfa y beta.
El segundo proyecto fue la creación de un sistema multiagentes donde se pudiera modelar las interacciones que existen en un safari 
dándole parámetros claves a los agentes para buscar soluciones reales.
